# EVOLVE 3 Study Guide (B1)

Plataforma interactiva para el libro EVOLVE 3 de Cambridge.  
Construida con React, Vite, Tailwind CSS y Gemini AI.

## Características

- 12 unidades completas (vocabulario, gramática, expresiones, writing, speaking, listening)
- Buscador global (vocabulario + gramática)
- Tarjetas de gramática con:
  - **Topic Quiz** (generado por IA con Gemini)
  - **Unit Quiz** (15 preguntas mezcladas)
- Evaluador de escritura con IA (feedback instantáneo)
- Diseño responsivo, modo oscuro, acento #eb515b

## Instalación

```bash
npm install
npm run dev
